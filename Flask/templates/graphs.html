Model Training Performance Graphs

Classification Report

A classification report is a concise summary that evaluates the performance of a classification model. It provides key metrics such as precision, recall, and F1 score, which indicate the model's accuracy in predicting different classes. By analyzing these metrics, users can assess the model's effectiveness in correctly classifying instances and identify areas for improvement.

Confusion Matrix

A confusion matrix is a tabular representation that summarizes the performance of a classification model by comparing predicted labels with actual labels. It consists of four main components: true positives (TP), true negatives (TN), false positives (FP), and false negatives (FN).

Data Distribution

A pie chart visually represents the composition of a whole by dividing it into sectors, each proportional to its corresponding value, providing an easy-to-understand depiction of relative proportions or percentages.

Roc Curve

A ROC (Receiver Operating Characteristic) graph illustrates the performance of a binary classification model across various thresholds by plotting the true positive rate (sensitivity) against the false positive rate (1 - specificity).
